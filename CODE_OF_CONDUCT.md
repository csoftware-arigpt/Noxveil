# Code of Conduct

## Our Standard

We want Noxveil to be a respectful, constructive project for authorized security research and lab work.

People participating in this project are expected to:

- be respectful in discussion, review, and feedback
- assume good intent and communicate clearly
- focus criticism on ideas, code, and process, not people
- welcome contributors with different backgrounds and experience levels
- keep discussions professional, especially when handling sensitive security topics

## Expected Behavior

Examples of behavior that help create a healthy project space:

- giving actionable, respectful feedback
- asking questions without hostility or ridicule
- documenting decisions and tradeoffs clearly
- reporting security concerns privately when appropriate
- keeping examples and contributions aligned with the project's educational and authorized-use scope

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, insults, or personal attacks
- discriminatory or hateful language
- intimidation, trolling, or deliberate disruption
- publishing private information without permission
- pressuring contributors to support unauthorized, harmful, or illegal use
- submitting content intended to facilitate abuse outside authorized lab or research environments

## Security and Responsible Use

Noxveil is intended for educational work, internal labs, and authorized security research.

By contributing, participating, or discussing changes here, you agree to keep the project aligned with that purpose.
Contributions or discussions that clearly promote unauthorized access, destructive use, credential theft, or real-world abuse may be removed and may result in suspension from project spaces.

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may:

- remove or edit comments, issues, or pull requests that violate this policy
- decline contributions that are incompatible with the project's scope or community standards
- temporarily or permanently restrict participation for repeated or serious violations

## Reporting

If you experience or witness behavior that violates this Code of Conduct, contact the project maintainers through a private channel available for the repository.

When possible, include:

- what happened
- where it happened
- who was involved
- any supporting context or screenshots

Reports will be reviewed as promptly and fairly as possible.

## Scope

This Code of Conduct applies to:

- repository discussions, issues, pull requests, and reviews
- project documentation and examples
- other official community spaces managed for Noxveil

## Consequences

Maintainers may apply consequences based on severity, including:

- a private warning
- removal of content
- temporary participation limits
- permanent removal from project spaces
